
# Enterprise SOC Home Lab Documentation

## Overview

This directory contains the complete documentation for the **Enterprise SOC Home Lab** project. Each section explains the deployment, configuration, attack simulation, and investigation process used to build a realistic Security Operations Center (SOC) environment.

The documentation is organized to follow the lifecycle of the lab, from infrastructure deployment to incident response and threat analysis.

---

## Documentation Structure

| Section | Description |
|---------|-------------|
| **01-Architecture** | Enterprise SOC network architecture and lab design |
| **02-Installation** | Installation and configuration of all virtual machines and services |
| **03-Active-Directory** | Domain Controller, DNS, Organizational Units, Users, and Group Policy |
| **04-Wazuh** | Wazuh Manager, Indexer, Dashboard, Agents, Rules, and Alerting |
| **05-Sysmon** | Sysmon deployment, configuration, logging, and event monitoring |
| **06-Attack-Simulation** | Simulated attack scenarios used to generate security events |
| **07-Incident-Response** | Alert investigation, evidence collection, and incident reports |
| **08-MITRE-Mapping** | MITRE ATT&CK mappings for validated attack simulations |

## Objectives

- Deploy a realistic enterprise SOC lab.
- Centralize endpoint logs using Wazuh SIEM.
- Monitor Windows events with Sysmon.
- Simulate attacker techniques.
- Investigate alerts generated by Wazuh.
- Map security events to the MITRE ATT&CK Framework.
- Develop practical SOC Analyst skills through hands-on experience.
