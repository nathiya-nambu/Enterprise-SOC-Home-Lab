# MITRE ATT&CK Mapping

The Enterprise SOC Home Lab uses the MITRE ATT&CK framework to classify simulated attack techniques.

| Attack | Technique | ID |
|---------|-----------|----|
| PowerShell | Command and Scripting Interpreter | T1059 |
| Failed Login | Valid Accounts | T1078 |
| Discovery Commands | System Information Discovery | T1082 |
| Registry Modification | Modify Registry | T1112 |
| File Creation | Ingress Tool Transfer | T1105 |
| EICAR | Malware Simulation | T1105 |

## Objective

Map security events to industry-standard attack techniques for investigation and threat hunting.
